# 100 Days DevOps Journal

Reto personal de 100 días para aprender y practicar DevOps. Cada día se documenta un desafío, concepto o práctica del mundo DevOps.

## Reglas del reto

1. Dedicar tiempo cada día a un tema de DevOps
2. Documentar lo aprendido con comandos, código y recursos
3. Registrar problemas encontrados y sus soluciones
4. Cada día se guarda con su propio README
